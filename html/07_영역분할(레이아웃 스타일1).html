레이아웃(layout)

    사전적 의미 : 배치, 정리

    기술적 의미 : 구성 요소를 제한된 공간에 효율적으로 배치하는 것

화면 배치(표시, 형식) 지정 : display 속성

    요소가 화면에 어떻게 표시(배치)될 지 형식을 지정하는 속성

    - block : 화면을 수직분할
              (세로 화면을 가로로, 레고)
              -> 행이 나뉘어짐
                  + 한 행을 전부 차지하려는 성질
                  + width, height 크기 지정 속성을 사용가능
               

    - inline : 화면을 수평분할
              (가로 화면을 세로로, 책꽂이)
              -> 한 행의 열이 나뉨
                  + 작성된 내용(content) 만큼의 크기를 차지
                  + width, height 사용 불가

    - inline-block : 화면 수평분할 + 크기 지정 가능

    - none : 화면에 표시 X
             (화면에 미표시, 존재는 하고 있음)

    - flex : 요소가 정렬되는 방향, 요소와의 간격을 유연하게 처리하여 표시
             -> 브라우저가 간격, 크기 등 계산하여 적용해 줌

    - grid : 격자무늬 형식으로 화면 쉽게 분할하는 형식
